# EV vs Gas TCO Calculator

A Next.js application for calculating and comparing the 5-year total cost of ownership (TCO) between Electric Vehicles (EV) and Gas vehicles.

## Features

- Interactive calculator with real-time TCO calculations
- Advanced options for resale values and ownership period
- Cost breakdown visualization
- Currency-agnostic formatting

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Zod (validation)
